# Парсер рассписаний
## Описание
Проект автоматически собирает данные из Airtable c использованием библиотеки Selenium, сохраняя в формате JSON и предоставляет API через DJango. Используется Docker-окружение.

### Требования
- Docker 20.10+
- Docker Compose 2.0+

### Запуск
```bash
docker-compose build
docker-compose up
